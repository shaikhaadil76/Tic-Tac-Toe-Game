# 🎮 Tic Tac Toe Game

A responsive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. This project allows two players to play Tic Tac Toe in the browser with a clean and modern user interface.

---

## 🚀 Features

- 🎯 Two-player gameplay
- 🏆 Winner detection
- 🤝 Draw detection
- 🔄 Reset Game button
- 🆕 New Game button
- 📱 Responsive design
- 🎨 Modern dark theme
- ⚡ Fast and interactive gameplay

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/tic-tac-toe-game.git
```

2. Open the project folder.

3. Double-click **index.html** or open it with Live Server in Visual Studio Code.
